Fitness Hub - Site Plan

Site Name

Fitness Hub – A personal fitness website dedicated to simple routines and tools that make exercising easier.

Reason: I chose this name because it’s short, memorable, and clearly communicates the site’s purpose as a central hub for fitness tips and routines.

Site Purpose

The site provides quick workout tips, simple exercise routines, and interactive tools like a random workout generator. It also includes a contact form for users to reach out. The purpose is to help beginners and busy individuals stay active with practical, easy-to-follow workouts.

Scenarios

  • “I have 15 minutes before work, what quick workout can I do?”
  • “How can I save my favorite exercises so I can use them later?”

Color Schema

Chosen colors:

  • #2E86AB (Blue): Used for headings and accents to convey energy and trust.
  • #F2A541 (Orange): Used for buttons and highlights to create motivation and draw attention.
  • #333 (Dark Gray): Used for body text to ensure readability.

Typography

Fonts chosen:

  • Roboto – for body text (easy to read on all devices).
  • Montserrat – for headings (modern and bold, fits fitness theme).
